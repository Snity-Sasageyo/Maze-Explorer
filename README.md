# Maze Explorer

It is an endless procedurally generated maze that builds itself one step ahead of you and deletes it. 

## Controls

| Key | Action |
| :--- | :--- |
| `W` / `↑` | Forward |
| `S` / `↓` | Backward |
| `A` / `←` | Left |
| `D` / `→` | Right |
| `Q` / `E` | Turn Left / Right |
| `ESC` | Pause Game |
